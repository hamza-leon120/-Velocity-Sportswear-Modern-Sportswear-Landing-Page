## ✨ Features

- 📱 Fully responsive — mobile-first design approach
- 🗂️ Modern grid-based product layout
- 📲 Animated mobile sidebar navigation
- 🔝 Scroll-to-top button
- 🎨 Smooth transitions and clean UI throughout
- 🧩 Modular SCSS architecture with variables and mixins
- 🏗️ Pug templating for clean, reusable HTML structure
- ⚡ Lightweight vanilla JavaScript for all interactions

## 📂 Project Structure

```
Velocity-Sportswear/
├── .github/
│   └── workflows/        # GitHub Actions CI/CD pipeline
├── css/                  # Compiled CSS output
├── img/                  # Images and screenshots
├── include/              # Pug partials (sections, components)
├── scss/                 # SCSS source files
│   ├── layout/           # Layout and grid styles
│   └── mixin/            # Reusable mixins
├── webfonts/             # Custom font files
├── LICENSE
├── README.md
├── index.html            # Compiled HTML output
├── index.pug             # Pug source template
└── main.js               # JavaScript interactions
```

## ⚙️ Getting Started

```bash
# 1. Clone the repository
git clone https://github.com/hamza-leon120/-Velocity-Sportswear-Modern-Sportswear-Landing-Page.git

# 2. Navigate into the project folder
cd -Velocity-Sportswear-Modern-Sportswear-Landing-Page

# 3. Open in your browser
open index.html
```

> **Note:** To edit the source files, modify `index.pug` and the `scss/` files. You'll need a compiler like [Prepros](https://prepros.io/) or a Gulp/Webpack setup to compile Pug → HTML and SCSS → CSS.
